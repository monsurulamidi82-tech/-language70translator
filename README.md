# Language70 Translator Bot 🤖

A Telegram bot that translates text and voice messages into 70+ languages.

## Features ✨

- Text translation to 70+ languages
- Voice message transcription and translation
- User language preferences
- Reply translation with `/tl` command
- Interactive language selection
- 24/7 deployment on Railway

## Commands 📝

- `/start` - Welcome message
- `/help` - Help menu
- `/setlang` - Set target language
- `/langlist` - List all languages
- `/about` - Bot information
- `/tl` - Translate replied message
