# Language70 Translator Bot

Telegram bot for translating text and voice into 70+ languages.

## Quick Deploy

1. Create bot with @BotFather, get token
2. Push to GitHub
3. Deploy on Railway
4. Add TELEGRAM_TOKEN environment variable

## Commands
- /start - Welcome
- /setlang - Choose language
- /langlist - List all languages
- /tl - Translate replied message
